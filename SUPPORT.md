# Support

Thank you for using NOVA projects. If you need help, have a question, or want to report an issue, here's the right channel to use.

---

## Start with the Docs

Before opening an Issue, check the [documentation](https://docs.novacont.tech). Most setup questions, usage instructions, and FAQs are covered there, and it's the fastest way to get an answer.

---

## GitHub Issues

Found a bug, or want to request a feature? Open a GitHub Issue using the appropriate template.

Please search existing Issues first to avoid duplicates.

---

## Security Issues

If your report involves a security vulnerability, **do not open a public Issue.**

Follow the instructions in [SECURITY.md](./SECURITY.md) to report it privately instead.

---

## Community

Join our [Discord](https://discord.gg/novacont) to ask quick questions, discuss ideas, and get project announcements. For anything that isn't a confirmed bug or a security report, Discord is usually the fastest way to reach us.

---

## Contact

| Purpose | Email |
|---|---|
| General questions, business inquiries | support@novatechnology.app |
| Security reports | security@novatechnology.app |

---

## Response Times

NOVA is built and maintained by a small team, so there's no guaranteed SLA for general support requests, we respond as time allows, typically within a few days. Security reports are the exception: those follow the fixed response targets in [SECURITY.md](./SECURITY.md).

If something is urgent, Discord is generally faster than email.
